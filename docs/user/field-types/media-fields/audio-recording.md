# Audio Recording

_How to add and configure an Audio Recording field in the {{Notebook}} Editor._

---

## What This Field Does

An Audio Recording field allows capture of an audio recording using the
device microphone. The audio recording will be saved as an attachment
on the record associated with the named field. The format
of the recording is device dependent. Use it to capture audio commentary
from the user or interview audio between the user and a third party.

**Note** that while recording is taking place, the recording controls
are shown at the bottom of the screen to pause, stop or cancel the
recording. During recording, the user cannot navigate away from
the current form page. They can continue to interact with the form
on that page however. They cannot navigate to another section
of the form if the form is configured to use [tabs for each section].

```{screenshot} field-types-design/audio-recording-controls.png
:alt: Audio Recording controls during a recording capture.

```

If you want to capture a recording while filling out a form, you
should put all of your form fields into one section or configure
the form to use _inline_ sections in the Form Settings.

## Adding the Field

To add this field, open the
[ADD A FIELD dialog](../shared-settings/adding-a-field.md), navigate to the
**MEDIA** tab, and click the **Audio Recording** card. Then click the
**ADD FIELD** button in the lower right.

## Configuring the Field

Click the field's **grey header bar** to expand it and see its settings.
For an overview of the settings shared by all fields — including Label,
Helper Text, Field ID, and the field toolbar — see
[Field Identity](../shared-settings/field-identity.md) and
[Field Toolbar](../shared-settings/field-toolbar.md).

Give the field a meaningful Label, review the auto-populated
Field ID, and add any desired Helper Text.

### Shared Field Options

Configure any of the shared field options as needed.

For settings shared across all field types — including Required,
Annotation, Uncertainty, Conditions, Copy value to new records,
and Display in child records — see
[Field Options](../shared-settings/field-options.md).
