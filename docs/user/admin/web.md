# Using the {{Dashboard}}

The {{dashboard}} is the web based control centre for {{FAIMS}} where you can manage
your teams and {{notebooks}} and where administrators can manage users.

When you log in to the {{FAIMS}} {{Dashboard}}, you'll see a left sidebar with one or two sections:

## Content

This section is only visible if you have permission to create or use {{notebooks}}. It would not
be visible if you only have the _Operations Administrator_ role.

- **{{Notebooks}}** — View and manage data collection {{notebooks}}
- **Templates** — View and manage reusable notebook templates

## Management

- **Users** — View all system users and their roles (admin users only)
- **Teams** — View and manage teams

> 💡 **Tip**: Click on any sidebar item to navigate to that section. The breadcrumb at the top (e.g., "Home > Users") shows your current location. By default, you will see the _Teams_ page.
