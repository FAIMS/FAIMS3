# Operating Systems

**{{FAIMS}}** supports the following operating systems:

1. Google Android 11+
2. iOS
3. Chrome   

# Hardware

There are no minimum specifications for hardware, however we recommend Android or iOS devices that started production in the last two years.

Updated: 11 February 2026

# Connected devices

**{{FAIMS}}** can collect data from some external devices connected via Bluetooth, including **external GPS** devices. For connections to other instrument data see [External Devices](./external-devices.md).
