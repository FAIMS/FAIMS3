Class SyncHandler<Content>

Translates 'paused', 'active', and 'change' events from a PouchDB to the events that events.ts works with.

This was created when the assumptions I made about pouch were wrong, Pouch should theoretically batch its updates, so in the future I hope to be able to remove this class.
